# Premix Scheduler

Aplicativo MVP en Python + Streamlit para secuenciar órdenes de producción de premezclas, minimizando riesgo de contaminación cruzada e indicando eventos de limpieza.

## Tecnologías
- Python
- Streamlit
- SQLite
- openpyxl
- pandas
